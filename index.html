Sobre Mim

Meu nome é Gustavo Nascimento

Sou formado em Física pela USP e estou à procura de um oportunidade para trabalhar com para Ciência de Dados.

Me formei como técnico em informática junto ao ensino médio pela ETEC de Sapopemba, onde eu aprendi lógica de programação, conceitos de orientação nas linguagens Java e Visual Basic, manipulação de bancos de dados com MySQL e Microsoft Access e etc. No inicio de 2018, eu iniciei minha graduação em Física pela Universidade de São Paulo e conclui o curso ao final de 2022.

Estou buscando uma oportunidade de trabalhar profissionalmente como Cientista de Dados para melhorar a tomada de decisão da empresa, através da construção de soluções usando dados.

Projetos em Ciência de Dados

Desenvolvimento de um Painel para Negócio com o Streamlit

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, juntos com ferramentas de desenvolvimento web como o Streamlit e o Github, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na India.

O resultado final do projeto foi um painel hospedado em um ambiente Cloud e disponibilizado através de um link web. O painel pode ser acessado por qualquer dispositivo com acesso a internet.

As ferramentas utilizadas foram:

  • Python.
  • Jupyter Notebook.
  • Terminal
  • Streamlit

Ensaio de Machine Learning

Nesse projeto, foram explorados os principais conceitos do treinamento, ajuste de parâmetros e limiares entre overfitting e underfitting de vários algoritmos de Machine Learning dentro das tarefas de classificação, regressão e agrupamento, através de um ensaio de Machine Learning.

O resultado final do algoritmo foi um painel mostrando performance desses algoritmos de Machine Learning, a partir das variações dos valores dos principais parâmetros de cada algoritmo, que controlam o estado de overfitting e underfitting.

As ferramentas utilizadas foram:

  • Python 3.10 e Scikit-learn
  • Algoritmos de classificação: KNN, Decision Tree, Random Forest e Logistic Regression.
  • Algoritmos de regressão: Linear Regression, Decision Tree Regressor, Random Forest Regressor, Polynomial Regression, Linear Regression Lasso, Linear Regression Ridge, Linear Regresion Elastic Net, Polynomial Regression Lasso, Polynomial Regression Ridge e Polynomial Regression Elastic Net..
    • Algoritmos de clusterização: K-Means e Silhouette Score
  • Métricas de performance: Accuracy, Precision, Recall, F1-Score, R2, MSE, RMSE, MAE, MAPE e Silhouette Score.
