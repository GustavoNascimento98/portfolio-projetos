Dashboard Airbnb Nova York PowerBI

Análise das Hospedagens em Nova York


Link para o Painel Online: Visão Geral

1. Visão Geral

O relatório desenvolvido nesse projeto examina o mercado de acomodações disponíveis na cidade de Nova York, utilizando dados do Airbnb fornecidos via Kaggle. O objetivo principal é identificar tendências e padrões nas características das propriedades listadas, como tipo de acomodação, localização, preço médio e avaliações dos usuários. A partir dessas informações, foram desenvolvidos dashboards no Power BI que oferecem insights valiosos para a tomada de decisões estratégicas, como ajustes de preços, melhorias na qualidade dos serviços e identificação de oportunidades de mercado.

2. Problema de Negócio

A Airbnb atua em um mercado dinâmico e altamente competitivo, que possui uma ampla variedade de acomodações com características distintas e uma demanda que está constantemente em evolução por parte dos viajantes. A falta de insights precisos sobre a situação do mercado dificulta a tomada de decisões estratégicas, como ajustes de preços, otimização de serviços e identificação de oportunidades de mercado.

3. Solução

O relatório desenvolvido nesse projeto usando a ferramenta Power BI oferece uma solução para esse problema ao fornecer uma visão abrangente e detalhada do mercado de acomodações em Nova York. Ele possui duas principais visões:

  1. Métricas Gerais: nessa página temos uma visão geral sobre o panorama do imóveis na cidade de Nova York. Aqui podemos verificar, dados sobre o número de reviews e o preço médio por bairro e distritos, além de outras informações.

  2. Evolução de Métricas: nessa página conseguimos verificar a variação de métricas conforme o tempo e também a localização geográfica dos imóveis.

Atenção: nessa análise assumimos as seguintes hipóteses:

  • A data de primeiro review de uma acomodação corresponde a data de inscrição do imóvel no airbnb pelo proprietário.
  • Todos os clientes fizeram uma avaliação sendo a data de avaliação correspondente a data de check-in (reviews = reservas).

4. Insights

Nesse relatório foi possível identificar os seguintes insights:

  1. A maior quantidade de reserva ocorrem no verão do hemisfério norte, no período de férias escolares durante os meses de Junho e Julho.

  2. O distrito de Manhattan é o mais caro para os 3 tipos de acomodações.

  3. Quartos compartilhados recebem média menos reviews.

5. Próximos Passos

Possíveis aprimoramentos para o relatório do Power BI incluem:

  1. Integração com fontes de dados em tempo real para insights atualizados.

  2. Derivar novas features para tornar a análise mais precisa.
